# Security Policy

## Supported Versions

Only the latest major/minor version of `myutils-deepgajera` receives security updates. We encourage all users to upgrade to the latest release to ensure they have the latest security patches.

| Version | Supported          |
| ------- | ------------------ |
| >= 0.0.2| :white_check_mark: |
| < 0.0.2 | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability in this project, please report it privately by emailing the maintainer:

* **Email:** deepgajera11@email.com
* **Subject:** `SECURITY VULNERABILITY: <brief description>`

Please include the following details in your report:

1. **Description**: A detailed description of the vulnerability and its potential impact.
2. **Steps to Reproduce**: A step-by-step guide or proof-of-concept (PoC) script to reproduce the issue.
3. **Environment**: Python version, OS version, and the version of `myutils-deepgajera` being used.

### Our Response Process

1. **Acknowledgment**: We will acknowledge receipt of your report within **48 hours** and provide an initial assessment of the issue.
2. **Investigation**: We will investigate the issue and coordinate a fix.
3. **Disclosure**: Once the fix is ready, we will publish a new release and disclose the vulnerability to the public, giving you full credit for the discovery (unless you prefer to remain anonymous).
